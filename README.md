## Смысл тестов

### Tests_sircle.py

Тесты отвечают за расчет площади и периметра круга

- `test_area_positive_radius` - проверка расчета площади круга положительном радиусе

- `test_area_negative_radius` - тест на вывод ошибки при отрицательном радиусе для площади

- `test_perimeter_positive_radius` - проверка расчета периметра круга при положительном радиусе

- `test_perimeter_negative_radius` - тест на вывод ошибки при отрицательном радиусе для периметра

### tests_triangle.py

- `test_positive_good_perimeter` - проверка расчета периметра треугольника с корректными положительными сторонами

- `test_positive_good_area` - проверка расчета площади треугольника с корректными положительными сторонами по формуле Герона

- `test_negative_good_perimeter` - тест на вывод ошибки при наличии отрицательной стороны для периметра

- `test_negative_good_area` - тест на вывод ошибки при наличии отрицательной стороны для площади

- `test_positive_bad_perimeter` - тест на вывод ошибки при некорректных сторонах треугольника (нарушение неравенства треугольника) для периметра

- `test_positive_bad_area` - тест на вывод ошибки при некорректных сторонах треугольника (нарушение неравенства треугольника) для площади

- `test_negative_bad_perimeter` - тест на вывод ошибки при наличии отрицательной стороны и некорректных сторонах треугольника для периметра

- `test_negative_bad_area` - тест на вывод ошибки при наличии отрицательной стороны и некорректных сторонах треугольника для площади

### tests_rectangle.py

- `test_two_positive_sides_area` - проверка расчета площади прямоугольника при двух положительных сторонах

- `test_one_zero_side_area` - проверка расчета площади прямоугольника при одной нулевой стороне

- `test_two_zero_sides_area` - проверка расчета площади прямоугольника при двух нулевых сторонах

- `test_one_negative_side_area` - тест на вывод ошибки при одной отрицательной стороне для площади

- `test_two_negative_sides_area` - тест на вывод ошибки при двух отрицательных сторонах для площади

- `test_two_positive_sides_perimeter` - проверка расчета периметра прямоугольника при двух положительных сторонах

- `test_one_zero_side_perimeter` - тест на вывод ошибки при одной нулевой стороне для периметра

- `test_two_zero_sides_perimeter` - тест на вывод ошибки при двух нулевых сторонах для периметра

- `test_one_negative_side_perimeter` - тест на вывод ошибки при одной отрицательной стороне для периметра

- `test_two_negative_sides_perimeter` - тест на вывод ошибки при двух отрицательных сторонах для периметра

### tests_square.py

- `test_positive_shapes_perimeter` - проверка расчета периметра квадрата при положительной стороне

- `test_negative_shapes_perimeter` - тест на вывод ошибки при отрицательной стороне для периметра

- `test_positive_shapes_area` - проверка расчета площади квадрата при положительной стороне

- `test_negative_shapes_area` - тест на вывод ошибки при отрицательной стороне для площади
